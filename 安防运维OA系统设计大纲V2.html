一、系统概述

1.1 项目背景与目标

安防行业具有项目分散、外勤频繁、设备种类多、售后维保要求高等特点。传统管理方式依赖纸质单据和微信群，存在信息不透明、流程不规范、数据难追溯等问题。本系统旨在打造一套面向安防行业的一体化OA办公平台，实现从项目立项到售后维保的全生命周期管控，支持多端协同办公，提升企业运营效率与服务质量。

核心目标：

  • 项目全流程数字化：立项→询价→合同→采购→施工→结算→售后，全程在线、可追溯
  • 售后服务闭环管理：报修→派单→维修→确认→汇总，提升客户满意度
  • 多端协同：PC端深度办公 + 移动端灵活外勤 + 微信小程序轻量互动
  • 业务财务一体化：项目成本自动归集，应收应付实时掌握
  • 知识资产沉淀：技术文档、项目经验形成企业知识库，新人快速上手

1.2 设计原则

原则说明
移动优先安防行业外勤多，移动端为核心使用场景，PC端为管理与分析中心
业务驱动以项目全流程与售后服务为主轴，所有模块围绕核心业务展开
数据贯通客户→项目→售后→库存→财务，数据一次录入、全链路流转
灵活配置审批流程、考勤规则、权限体系均可自定义，适配不同规模企业
安全可控数据加密传输、操作日志审计、敏感字段权限隔离
渐进实施模块化设计，分阶段上线，降低实施风险

二、多端架构设计

安防行业的工作场景覆盖办公室、施工现场、客户现场、外出拜访等，不同场景对终端的需求差异显著。系统采用"一套后端、多端适配"的架构策略，确保数据一致性的同时，针对各终端特性优化交互体验。

2.1 PC客户端（Windows / macOS）

定位：深度办公与管理中心，适合管理人员、财务人员、项目内勤等在办公室内使用。

特性说明重点功能
技术方案Electron 框架，一套代码同时支持 Windows 和 macOS离线数据缓存、本地打印
交互优势大屏操作空间，复杂数据表格、多窗口并排、拖拽操作项目甘特图、报表分析、批量导入导出
系统集成调用本地打印机、本地文件系统、系统通知合同打印、CAD图纸预览、本地文件上传
消息推送系统级桌面通知、任务栏提醒审批待办、工单派发、到期预警
离线支持关键数据本地缓存，断网可查看，联网自动同步施工日志、客户资料离线查看

2.2 Android客户端

定位：外勤与施工现场首选，适合施工人员、售后工程师、外勤销售等。

特性说明重点功能
技术方案Flutter 跨平台框架，与 iOS 共享业务逻辑层代码热更新、插件化架构
GPS与定位高精度GPS定位、地理围栏、轨迹记录外勤打卡、施工位置签到、客户导航
相机集成直接调取手机相机拍照、录像，自动添加时间水印与GPS信息施工记录拍照、维修过程拍照、设备巡检拍照
推送通知集成华为/小米/OPPO/VIVO等厂商推送通道工单派发、审批提醒、紧急报警
NFC/蓝牙支持NFC标签读取、蓝牙设备连接设备扫码巡检、蓝牙门禁调试

定位：管理层与销售首选，注重审批效率与客户管理。

  • 施工变更与签证管理

  • 项目利润分析（收入-成本-费用）
  • 质保期设定与到期提醒
  • 项目档案自动归档（合同/图纸/验收资料/照片等）

3.5.7 质保管理

  • 质保期自动计算（从验收日期起算）

  • 质保期内免费维修工单自动关联
  • 转维保项目管理（续签维保合同）

  • 关联客户与项目（自动带出设备清单与历史维修记录）

  • 智能派单：根据故障类型、客户位置、人员技能标签、当前负载自动推荐最佳维修人员
  • 移动端推送通知（APP推送 + 微信模板消息）

3.6.3 维修进度

  • 工单状态实时更新（待处理→已派单→维修中→待确认→已完成）
  • 维修人员GPS签到/签退（到达客户现场自动签到）
  • 维修过程拍照记录（移动端拍照，自动加水印）
  • 备件更换记录（关联库存，自动扣减）
  • 超时预警与自动升级（SLA未达标时通知主管）
  • 客户可通过微信小程序实时查看维修进度

3.6.4 维修单据

  • 维修确认单（移动端手写签名，客户现场确认）
  • 维修费用清单（材料费 + 人工费 + 其他费用）
  • 更换备件清单与库存自动扣减
  • 客户评价与满意度评分（微信小程序评价）
  • 电子签章与PDF导出（PC端生成正式单据）

3.6.5 单据汇总

  • 按客户/单位汇总维修次数与费用
  • 按设备类型统计故障率（辅助产品选型）
  • 按维修人员统计工单量、响应时效与客户评分
  • 月度/季度维修费用汇总报表
  • 维保合同执行情况统计

3.6.6 维保合同与预防性维护

  • 年度维保合同签订、到期续签提醒
  • 合同约定巡检计划自动生成（月度/季度/半年度）
  • 预防性维护：根据设备使用年限与类型，自动生成定期巡检计划
  • 客户自助门户（微信小程序）：在线报修、查看进度、确认完工、评价服务
