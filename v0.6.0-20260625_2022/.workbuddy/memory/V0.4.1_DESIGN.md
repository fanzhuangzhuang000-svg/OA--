# V0.4.1 项目预算设计稿

3 表 + 7 API + 4 UI + 1 调度 + 90%/100% 双拦截

## 1. 数据库（3 张新表）

### 1.1 project_budgets 主表
```sql
CREATE TABLE project_budgets (
  id BIGSERIAL PRIMARY KEY,
  project_id BIGINT NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
  code VARCHAR(32) UNIQUE NOT NULL,
  version INT NOT NULL DEFAULT 1,
  status VARCHAR(16) NOT NULL DEFAULT 'draft',
  material_budget   DECIMAL(15,2) DEFAULT 0,
  labor_budget      DECIMAL(15,2) DEFAULT 0,
  outsource_budget  DECIMAL(15,2) DEFAULT 0,
  other_budget      DECIMAL(15,2) DEFAULT 0,
  total_budget      DECIMAL(15,2) DEFAULT 0,
  material_actual   DECIMAL(15,2) DEFAULT 0,
  labor_actual      DECIMAL(15,2) DEFAULT 0,
  outsource_actual  DECIMAL(15,2) DEFAULT 0,
  other_actual      DECIMAL(15,2) DEFAULT 0,
  total_actual      DECIMAL(15,2) DEFAULT 0,
  approved_by BIGINT REFERENCES users(id),
  approved_at TIMESTAMP NULL,
  created_by  BIGINT NOT NULL REFERENCES users(id),
  created_at  TIMESTAMP NOT NULL DEFAULT NOW(),
  updated_at  TIMESTAMP NOT NULL DEFAULT NOW(),
  remark TEXT
);
CREATE INDEX idx_pb_project ON project_budgets(project_id, status);
```

### 1.2 project_budget_items 明细
```sql
CREATE TABLE project_budget_items (
  id BIGSERIAL PRIMARY KEY,
  budget_id BIGINT NOT NULL REFERENCES project_budgets(id) ON DELETE CASCADE,
  category VARCHAR(16) NOT NULL,
  item_name VARCHAR(128) NOT NULL,
  specification VARCHAR(255),
  unit VARCHAR(16),
  quantity DECIMAL(10,2) NOT NULL DEFAULT 1,
  unit_price DECIMAL(12,2) NOT NULL DEFAULT 0,
  planned_amount DECIMAL(15,2) NOT NULL DEFAULT 0,
  item_id BIGINT,
  item_type VARCHAR(32),
  remark TEXT,
  sort_order INT NOT NULL DEFAULT 0,
  created_at TIMESTAMP NOT NULL DEFAULT NOW(),
  updated_at TIMESTAMP NOT NULL DEFAULT NOW()
);
CREATE INDEX idx_pbi_budget ON project_budget_items(budget_id, category);
```

### 1.3 project_actual_costs 实际成本
```sql
CREATE TABLE project_actual_costs (
  id BIGSERIAL PRIMARY KEY,
  project_id BIGINT NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
  source_type VARCHAR(32) NOT NULL,
  source_id BIGINT NOT NULL,
  category VARCHAR(16) NOT NULL,
  amount DECIMAL(15,2) NOT NULL,
  cost_date DATE NOT NULL,
  description VARCHAR(255),
  metadata JSONB,
  created_at TIMESTAMP NOT NULL DEFAULT NOW(),
  updated_at TIMESTAMP NOT NULL DEFAULT NOW(),
  UNIQUE(source_type, source_id, category)
);
CREATE INDEX idx_pac_project ON project_actual_costs(project_id, cost_date);
```

## 2. 实时触发（3 个 Observer）

- StockRecord created (type=in)  → material/purchase_in
- StockRecord created (type=out) → material/stock_out
- ExpenseClaim updated (status=approved) → 按 category 分类 (人工/labor, 外包/outsource, 其他/other)

## 3. 7 个 API

| # | Method | Path | 权限 |
|---|---|---|---|
| 1 | GET | /api/construction/budgets | 项目可见 |
| 2 | GET | /api/construction/budgets/summary/{projectId} | 项目可见 |
| 3 | POST | /api/construction/budgets | 项目经理+ |
| 4 | GET | /api/construction/budgets/{id} | 项目可见 |
| 5 | PUT | /api/construction/budgets/{id} | 创建人 (draft) |
| 6 | POST | /api/construction/budgets/{id}/approve | 财务总监+ |
| 7 | POST | /api/construction/budgets/{id}/revise | 项目经理+ |

## 4. 90% / 100% 拦截

- 90% → BudgetWarning 事件 → 站内消息 + 邮件
- 100% → BudgetNotExceeded Rule → 写单据时硬阻断 (财务总监可强制提交)
