## 修了 2 个 bug
1. **`whereHas('roles', fn($q) => $q->where('name', 'admin'))`** — `where('name')` 被外层 query builder 解析成对 users 的 where, 误生成 `where 'has'=roles`. 改用 join `model_has_roles + roles` 直接查
2. **PG 序列滞后 23505 duplicate key** — 老的非 Laravel 通道 INSERT 滞后了 users_id_seq, 在 step3 入口加 `setval(pg_get_serial_sequence('users','id'), MAX(id))` 修复
