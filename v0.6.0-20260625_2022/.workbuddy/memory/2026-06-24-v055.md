## 踩坑
### WorkOrderPriority use 漏 import
V0.5.5 controller 用 `$wo->priority->toSeverity()` 但没 import enum,
PHP 解析成 `App\Http\Controllers\Api\WorkOrderPriority` → Class not found 500
**修**: 加 `use App\Enums\WorkOrderPriority;`

### nextCode race condition
原用 sequence 的 orderByDesc('id').value('code') + 解析, 删工单后 sequence 滞后 → 撞 unique
**修**: 用 `COALESCE(MAX(CAST(SUBSTRING(code FROM 'WO[0-9]{4}-([0-9]+)') AS INTEGER)), 0) + 1`
跨 2 个 controller 都改了
