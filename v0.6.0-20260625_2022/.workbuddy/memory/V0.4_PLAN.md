## 核心架构决策

### 实际成本计算（V0.4.1）
- **采购入库 / 领料出库 / 报销审批通过** → 实时触发 INSERT project_actual_costs

## 关键业务规则

### 质保期 = 只读镜像
- `warranty_service_records` 不允许直接 INSERT/UPDATE
- 唯一数据源 = `service_orders`（售后维修工单）
- `service_orders` Observer 自动同步
- 客户发起报修 → 走售后工单流程 → 完工时自动镜像到质保期表

### 往来资金 = 实时聚合视图
- supplier_ledger / customer_ledger 不建表
- 用 SQL 实时聚合 supplier_payables / customer_receivables
- V1.0 暂不做对账单 PDF
- V1.1 再加 PDF 导出

### 供应商账号 = 复用 users 表
- users.type = 'supplier'
- users.supplier_id → suppliers.id
- 路由级 RBAC 中间件 SupplierOnly / SupplierScope
- 禁止看任何 /project/* /employee/* /finance/* 等内部页面
