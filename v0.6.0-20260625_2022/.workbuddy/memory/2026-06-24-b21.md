# B2.1 完工 — 2026-06-24 21:15

## 交付内容 (5 脚本 + 1 cron + 1 logrotate)

### 1. 备份 (oa_daily_backup.sh, 88 行)
- 每日 02:00 自动跑 (cron)
- pg_dump 整库 → db_YYYYMMDD_HHMM.sql.gz
- tar 打包代码 (排除 vendor/node_modules/storage 子目录) → code_*.tar.gz
- MANIFEST.md 标记
- 滚动 7 天, 过期自动删
- 失败时写告警到 /tmp/oa-alert-pending

### 2. 监控告警采集 (oa_alert.sh, 92 行)
- 每 15min 跑
- 7 维度检查:
  - 磁盘 % (>80% warn, >90% crit)
  - 内存 % (>80% warn, >90% crit)
  - CPU load5 (>cores*2 warn)
  - 4 服务存活 (php-fpm/nginx/pg/redis)
  - /api/health 200
  - 备份新鲜度 (25h 内必须有)
  - 证书有效期 (14/30 天分级)
- 写告警 → /tmp/oa-alert-pending

### 3. 告警发送 (oa_alert_send.sh, 50 行)
- 每 5min 跑
- 15min 冷却 (md5 哈希去重, 同样内容不发 2 次)
- 优先 msmtp → mail → 仅写日志 (3 级 fallback)

### 4. 证书续期 (oa_cert_renew.sh, 38 行)
- 每日 03:00 跑
- 检测证书剩余天数 (<30 天触发)
- certbot renew --nginx
- 失败时入告警队列

### 5. 加固 (oa_harden.sh, 86 行)
- APP_DEBUG=false 强校验
- PHP display_errors=Off
- storage 权限 www-data:www-data
- UFW 启用 + 22/80/443/8081 白名单
- .env 保护建议

### Cron 设置 (5 条)
```
0 2 * * *    oa_daily_backup.sh    # 每日 02:00 full 备份
*/15 * * * * oa_alert.sh           # 监控告警采集
*/5 * * * *  oa_alert_send.sh      # 告警发送 (15min 去重)
0 3 * * *    oa_cert_renew.sh      # 证书续期检查
0 */6 * * *  schedule:run          # Laravel 调度兜底
```
